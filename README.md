# Student Grade Tracker 🎓

A Java console application designed to manage and track student grades efficiently[cite: 1]. Developed as part of the Java Programming Internship at **CodeAlpha**[cite: 1].

## 🌟 Features
* **Dynamic Data Storage:** Uses `ArrayList` to store student objects dynamically[cite: 1].
* **Input Validation:** Ensures entered grades are within valid numerical ranges (0 - 100).
* **Summary Analysis:** Automatically computes:
  * Average score across all students[cite: 1].
  * Highest scoring student[cite: 1].
  * Lowest scoring student[cite: 1].
* **Report Generation:** Displays a clean summary table of all recorded student scores[cite: 1].

* **Concepts Used:** Object-Oriented Programming (OOP), `ArrayList`, Data Structures, Scanner Input Handling[cite: 1]
